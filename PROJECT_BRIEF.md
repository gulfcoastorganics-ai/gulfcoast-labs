# Project Brief

## Goal

Create a premium, lightweight portfolio and studio site that can support capital raising and client acquisition.

## Brand Positioning

- GulfCoast Labs is the personal/operator brand.
- ZeroChill Co. must be shown clearly as client work for Danny Ford.
- Vestra Intel is an owned brand/project.
- Lifepvth is an owned brand/project.

## Design Direction

- Premium dark UI
- Gulf Coast meets cyber-operator aesthetic
- Technical, polished, and not childish
- Mobile-first
- Fast and lightweight

## Required Pages

- `/`
- `/work`
- `/services`
- `/labs`
- `/contact`

## Required Content

- Hero statement
- Services list
- Portfolio
- Service tiers
- Process steps
- Contact CTA with localStorage-only form

## Build Constraints

- Vite
- React
- Tailwind v4
- No TypeScript
- No Docker
- No Playwright
- No heavy animation libraries
